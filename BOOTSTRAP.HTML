O Bootstrap-u

Bootstrap je besplatan i otvoren izvor CSS okvir usmjeren na responzivan, prvo za mobilne uređaje izrada web stranica s prednjim krajem .Sadrži CSS - i (opcionalno) predloške dizajna temeljene na JavaScript za tipografiju , obrasce , tipke ,navigacija i ostale komponente sučelja.Bootstrap, izvorno nazvan Twitter Blueprint, razvili su Mark Otto i Jacob Thornton na Twitter kao okvir za poticanje dosljednosti unutarnjih alata. Prije Bootstrapa, razne su se knjižnice koristile za razvoj sučelja, što je dovelo do nedosljednosti i velikog tereta održavanja.

Boostrap 1-5

Bootstrap je mrežni okvir koji se fokusira na pojednostavljivanje razvoja informativnih web stranica (za razliku od web aplikacija ). Primarna svrha dodavanja u web projekt je primijeniti Bootstrapov odabir boje, veličine, fonta i izgleda na taj projekt.

JavaScript

JavaScript je skriptni programski jezik, koji se izvršava u web pregledniku na strani korisnika. Napravljen je da bude sličan Javi, zbog lakšega korištenja, ali nije objektno orijentiran kao Java, već se temelji na prototipu i tu prestaje svaka povezanost s programskim jezikom Java.

HTML-CSS

HTML je kratica za HyperText Markup Language, što znači prezentacijski jezik za izradu web stranica. Hipertekst dokument stvara se pomoću HTML jezika. HTML jezikom oblikuje se sadržaj i stvaraju se hiperveze hipertext dokumenta HTML je jednostavan za uporabu i lako se uči.
