JavaScript

JavaScript se često skraćuje kao JS, programski je jezik koji je usklađen sa specifikacijom ECMAScript.JavaScript je na visokoj razini, često se sastavlja pravovremeno i ima više paradigmi. Ima sintaksu u zagradama, dinamičko tipkanje, orijentaciju objekta zasnovanu na prototipu i prvorazredne funkcije.

Uz HTML i CSS, JavaScript je jedna od osnovnih tehnologija World Wide Weba. Više od 97% web stranica koristi ih na strani klijenta za ponašanje web stranica, često uključujući knjižnice trećih strana. Većina web preglednika ima namjenski JavaScript mehanizam za izvršavanje koda na korisnikovom uređaju.

Kao jezik s više paradigmi, JavaScript podržava stilove programiranja upravljane događajima, funkcionalne i imperativne. Ima programska sučelja za aplikacije (API) za rad s tekstom, datumima, regularnim izrazima, standardnim strukturama podataka i objektnim modelom dokumenta (DOM).

Standard ECMAScript ne uključuje nikakve ulaze/izlaze (I/O), kao što su umrežavanje, pohrana ili grafičke mogućnosti. U praksi, web preglednik ili drugi sustav za vrijeme izvođenja pruža JavaScript API -je za I/O. JavaScript motori izvorno su se koristili samo u web preglednicima, ali su sada jezgrene komponente drugih softverskih sustava, ponajviše poslužitelja i raznih aplikacija. Iako postoje sličnosti između JavaScripta i Jave, uključujući naziv jezika, sintaksu i odgovarajuće standardne knjižnice, dva su jezika različita i uvelike se razlikuju u dizajnu.
